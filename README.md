# Amazon FBA Corporate Website

A modern and responsive corporate website template for Amazon FBA, digital services, business solutions, and professional training programs.

## Features

- Responsive navigation and mobile menu
- Modern orange and navy design
- Hero, About, Services, and Courses sections
- Call-to-action and contact sections
- Contact form validation
- Active navigation on scroll
- Dynamic copyright year

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Google Fonts
- Font Awesome
